# Neovim 高级快捷键与技巧

## 常用技巧

### 光标移动

| 快捷键 | 说明                                                             |
| ------ | ---------------------------------------------------------------- |
| `(`    | 跳到句子开头                                                     |
| `)`    | 跳到句子末尾                                                     |
| `{`    | 跳到段落开头                                                     |
| `}`    | 跳到段落末尾                                                     |
| `[i`   | 跳转到 scope 的开头（如 bash 中从 `done` 跳到 `for`）            |
| `]i`   | 跳转到 scope 的末尾（如 bash 中从 `for` 跳到 `done`）            |
| `gi`   | 跳转到上次修改过的地方（注意：经常用 `A` 和 `I` 可能会打乱标记） |

### 编辑技巧

| 快捷键               | 说明                      |
| -------------------- | ------------------------- |
| `S`                  | 适合用于复制大段内容      |
| `yR<search><label>p` | 利用 `R` 的远程模式来复制 |
| `&`                  | 再次执行上一次的替换命令  |
| `v` 选中后 `o`       | 在左右端跳转              |

### 多光标编辑

| 快捷键               | 说明                                |
| -------------------- | ----------------------------------- |
| `<c-v>I` 或 `<c-v>A` | 块模式下使用 I 或 A，有多光标的效果 |

### 宏录制

| 快捷键 | 说明                                                                       |
| ------ | -------------------------------------------------------------------------- |
| `q`    | 录制宏（实际是录制在寄存器里，可以打印寄存器内容修改后再 yank 进该寄存器） |
| `@@`   | 回放最近使用的寄存器                                                       |
| `Q`    | 回放最新录制的寄存器                                                       |

### 跳转

| 快捷键 | 说明                                                                                       |
| ------ | ------------------------------------------------------------------------------------------ |
| `gi`   | 跳转到上次修改过的地方（如果被 `A` 和 `I` 打乱，回到 normal 模式用 `g;` 或 `g,` 继续跳转） |
| `g;`   | 跳转到上一个修改位置                                                                       |
| `g,`   | 跳转到下一个修改位置                                                                       |

### 其他

`/joe/e`: 搜索“joe”并保持光标在末尾

`3/joe/e`: 搜索第3个“joe”并保持光标在末尾

## 寄存器系统

### 系统寄存器

| 寄存器 | 说明                                            |
| ------ | ----------------------------------------------- |
| `+`    | 系统剪贴板（通过 `Ctrl+C` 复制，`Ctrl+V` 粘贴） |
| `*`    | 选择剪贴板（鼠标选中的文本，鼠标中键粘贴）      |

### 内部寄存器

| 寄存器 | 说明                                                             |
| ------ | ---------------------------------------------------------------- |
| `"`    | 无名寄存器                                                       |
| `_`    | 黑洞寄存器（删除的内容不会保存）                                 |
| `-`    | 小删除寄存器（删除少于一整行的内容保存到这里，避免覆盖主寄存器） |
| `=`    | 表达式寄存器（能够输出计算结果）                                 |
| `/`    | 最近一次搜索的内容                                               |
| `:`    | 上一次输入的命令                                                 |
| `0`    | 上一次 yank 的内容                                               |

### 使用方法

先指定寄存器，然后执行操作：

```text
# 先指定一个名为 r 的寄存器
"r
# 复制当前行
yy
# 后续任何时候都可以调用该寄存器

# 如果想要补充该寄存器里的内容可以使用
# 对应寄存器的大写字母
"R
# 复制另外的一行
yy
```

## 高级命令

### norm 命令

`norm` 命令可以在每一行执行 normal 模式的命令。

假设文件中只有以下内容：

```text
foo
Bar
fizz buzz
one two three
```

在命令行输入：

```bash
%norm IHELLO <C-v><Esc>lgUlea BEAUTIFUL<C-v><Esc>A WORLD
```

> 注意：这里的 `<C-v><Esc>` 是指需要在键盘上按下 `<C-v>` 再按下 `<Esc>`，意在告诉 norm 命令这里需要执行 `<Esc>`。`<C-v>` 就是告诉 neovim 后面敲下的 `<Esc>` 是一个特殊字符，需要转义。最终屏幕上显示的可能是 `%norm IHELLO ^[lgUlea BEAUTIFUL^[A WORLD`

文件中内容将替换为：

```text
HELLO Foo BEAUTIFUL WORLD
HELLO Bar BEAUTIFUL WORLD
HELLO Fizz BEAUTIFUL buzz WORLD
HELLO One BEAUTIFUL two three WORLD
```

## 实用案例

### 快速生成 100 行递增数字

在 Neovim 中利用 `Ctrl-a`（增加数字）的特性，配合宏或 Visual 模式的高级特性，可以非常优雅地完成这个任务。

#### 方法一：宏录制 + `Ctrl-a`（最符合直觉的做法）

1. 输入 `1`，按下 `Esc` 退回普通模式
2. **录制宏**：
   - 按 `qq` 开始录制宏（存入寄存器 q）
   - 按 `yyp`（复制当前行并粘贴到下一行）
   - 按 `Ctrl-a`（将新行的数字加 1）
   - 按 `q` 停止录制
3. **重复执行**：
   - 输入 `98@q`（因为已经有 1 和 2 了，还差 98 个）

#### 方法二：Visual Block 序列递增（最强"Vim味"做法）

Vim/Neovim 有一个专门针对多行数字生成序列的命令 `g <Ctrl-a>`。

1. **先生成 100 行 `1`**：
   - 输入 `100i1` 然后按 `Enter`，再按 `Esc`
   - _解释：这会插入 100 行，每行都是 1_
   - _或者：写一个 1，然后 `yy99p`_
2. **全选这 100 行**：
   - 按 `ggVG`（或者 `Ctrl-v` 选中这一列）
3. **生成序列**：
   - 按下 **`g`** 然后按 **`Ctrl-a`**

> **原理**：普通的 `Ctrl-a` 是把选中的所有数字都加同一个值；而 `g Ctrl-a`（Global Increment）会把选中的数字变成一个**递增序列**（1, 2, 3, ...）。

#### 方法三：命令行模式（最快、最简洁）

如果你不想动脑子录宏，直接用内置函数生成是最快的。

在普通模式下输入以下命令并回车：

```vim
:put =range(1, 100)
```

> **注意**：这会在当前光标下方插入 1 到 100。如果你是在空文件里，第一行可能是空行，最后删掉第一行即可（`ggdd`）。
