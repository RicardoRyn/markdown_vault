# 安装

去官网下载Homebrew，直接在终端键入：

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

下载完成后，终端会提示你：

```bash
==> Next steps:
- Run these commands in your terminal to add Homebrew to your PATH:
    echo >> /home/ricardo/.bashrc  # 键入
    echo 'eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"' >> /home/ricardo/.bashrc  # 键入
    eval "$(/home/linuxbrew/.linuxbrew/bin/brew shellenv)"  # 键入
- Install Homebrew's dependencies if you have sudo access:
    sudo apt-get install build-essential  # 键入
  For more information, see:
    https://docs.brew.sh/Homebrew-on-Linux
- We recommend that you install GCC:
    brew install gcc  # 键入
- Run brew help to get started
- Further documentation:
    https://docs.brew.sh
```
